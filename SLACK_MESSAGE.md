## Technical Details

**Built with:**
- React + TypeScript
- Tailwind CSS + shadcn/ui
- Recharts for data visualization
- Framer Motion for animations

**To run locally:**
```bash
cd foodieclub-dashboard
npm install
npm run dev
```

**To build for production:**
```bash
npm run build
```

**To deploy:**
The dashboard can be deployed to Vercel, Netlify, or any static hosting service. Just run `npm run build` and upload the `dist` folder.
